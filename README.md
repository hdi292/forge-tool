# Forge Tool — أدوات PDF

موقع أدوات PDF مجانية أونلاين، أكثر من 25 أداة في مكان واحد.
يعمل بالكامل في المتصفح بدون رفع الملفات لأي سيرفر.

## الملفات
- `index.html` — الصفحة الرئيسية
- `styles.css` — التنسيق
- `script.js` — السكريبت
- `robots.txt` — لمحركات البحث
- `sitemap.xml` — خريطة الموقع
- `llms.txt` — لمحركات الـ AI

## النشر على GitHub Pages
1. ارفع كل الملفات للـ repo
2. Settings → Pages → اختر `main` branch → Save
3. انتظر دقيقة، الرابط هيكون: `https://USERNAME.github.io/forge-tool/`
